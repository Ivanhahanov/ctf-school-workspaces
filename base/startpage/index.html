CTF School — Workspace

Это твоё изолированное рабочее окружение. Доступа в интернет нет — всё решается здесь.

Как решать

  1. Открой адрес сервиса задания (он указан в карточке таска на платформе).
  2. Инструменты — на рабочем столе: Firefox, Терминал, Файлы.
  3. Нашёл флаг — отправь его в карточке задания на платформе.

Буфер обмена

  • Скопировать наружу: выдели → Ctrl+C, затем открой буфер (иконка 📋 на панели слева) — текст появится там.
  • Вставка в окне: Ctrl+V
  • Вставка в терминале: Ctrl+Shift+V

Правила

  • Флаг уникален для твоей команды — делиться бесполезно.
  • Окружение одинаковое у всех: побеждает навык, не инструмент.
  • Решения через внешний AI — против духа соревнования.

Подсказки

  • Терминал: curl, vim, python3 уже есть.
  • Файлы задания (если выданы) лежат в твоём домашнем каталоге ~/.
  • Окно маленькое? Растяни браузер — рабочий стол подстроится.
